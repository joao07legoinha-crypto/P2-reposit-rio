# Alínea 4 - Tabela dos Scripts PHP

| Nome | Descrição | User Story |
|---|---|---|
| db.php | Estabelece a ligação à base de dados SQLite3 | USR0 |
| init_db.php | Cria a base de dados e as tabelas necessárias | USR0 |

| novareserva.php | Guarda os dados da reserva na base de dados | USR3 |
| acessos.php | Lista os registos de acesso dos utilizadores | USR4 |
